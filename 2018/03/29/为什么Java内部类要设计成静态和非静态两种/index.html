为什么Java内部类要设计成静态和非静态两种

发表于 2018-03-29

​ 静态内部类的作用：只是为了降低包的深度，方便类的使用，静态内部类适用于包含类当中，但又不依赖与外在的类，不用使用外在类的非静态属性和方法，只是为了方便管理类结构而定义。在创建静态内部类的时候，不需要外部类对象的引用。非静态内部类有一个很大的优点：可以自由使用外部类的所有变量和方法。
